# Documentation Index

This directory contains project-level documentation for the `game-forge` monorepo.

## Documents

- `project-structure.md`: repository layout, package responsibilities, and dependency boundaries
- `development-notes.md`: development workflow, commands, rules, and important cautions

## Who This Is For

- New contributors who need a quick map of the repository
- Coding agents that need repository context before making changes
- Maintainers who want one stable place to extend project guidance
